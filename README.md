# 🚀 Interactive Quiz App

An interactive and responsive **Quiz Application** built with **HTML, CSS, and Vanilla JavaScript**. This project allows users to test their knowledge through multiple-choice questions while practicing **DOM Manipulation**, **Event Listeners**, and **JavaScript Logic**.

## 🌟 Features

- 📝 Multiple-choice questions
- ✅ Instant answer selection
- 📊 Score tracking system
- ⏭️ Next question navigation
- 🎉 Final score screen
- 🔄 Restart quiz functionality
- 📱 Fully responsive design
- ⚡ Smooth and interactive user experience

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/interactive-quiz-app.git
```

### Navigate to the Project Folder

```bash
cd interactive-quiz-app
```

### Run the Project

Open the `index.html` file in your browser or use **Live Server** in VS Code.

---

## 🎮 How to Play

1. Start the quiz.
2. Read each question carefully.
3. Select the correct answer.
4. Move to the next question.
5. View your final score.
6. Restart the quiz and try again!
